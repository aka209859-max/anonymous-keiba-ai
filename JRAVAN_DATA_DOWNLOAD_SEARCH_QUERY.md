#### 4.3 サンプルコード
```python

import jvlink  # または該当するSDK

# 認証
api_key = "YOUR_API_KEY"
jv = jvlink.initialize(api_key)

# データ取得（2020年1月1日〜2025年12月31日）
start_date = "2020-01-01"
end_date = "2025-12-31"

data = jv.get_race_data(
    start_date=start_date,
    end_date=end_date,
    venues=["東京", "中山", "阪神", "京都", "中京", "新潟", "福島", "札幌", "函館", "小倉"]
)

# CSV保存
data.to_csv("data/jravan/raw/jra_2020_2025_all.csv")
```
